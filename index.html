Parameter set description

Sequence notation: Am, Cm, Gm and Um represent 2′-MOE-A, 2′-MOE-C, 2′-MOE-G and 2′-MOE-U, respectively. Legacy H/J/K/L notation is also accepted. DNA residues should be written as dA, dC, dG and dT.

RRHD: one strand is fully 2′-MOE modified and hybridized with RNA.

ALT: one strand contains partial/intermittent 2′-MOE modifications and is hybridized with RNA.

DNA/RNA: DNA strand hybridized with RNA.

RNA/RNA: unmodified RNA duplex.

Gapmer: both terminal wing regions are 2′-MOE modified, whereas the central gap region is DNA. MOE/MOE regions are calculated using RRHD parameters, DNA/RNA regions using DNA/RNA hybrid parameters, and MOE–DNA junctions using ALT-based approximation.

Gapmer priority: RRHD local MOE/MOE → DNA/RNA hybrid → ALT junction approximation → RNA/RNA → fallback.

Non-crowding condition: 100 mM NaCl, 10 mM Na2HPO4 (pH 7.0 at 37 °C), and 1 mM Na2EDTA.

Cell-like condition: 40 wt% PEG200, 100 mM NaCl, 10 mM Na2HPO4 (pH 7.0 at 37 °C), and 1 mM Na2EDTA.

Backbone limitation: the current calculator uses phosphate diester (P=O) parameters. Phosphorothioate (P=S) backbone effects are not yet included and are temporarily approximated using P=O parameters.

For equivalent orientation, the calculator searches direct, reversed, strand-swapped, and reverse-strand-swapped forms of each NN pair.

Self-complementary correction is applied only when Strand A itself is self-complementary, for example 5′-UGCAUGCA-3′.
